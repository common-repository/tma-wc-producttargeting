TMA-WC-ProductTargeting is a great way to improve your selling. Products are delivered for your customers segments.

== Description ==

The addon needs TMA-WebTools the integration for http://thorstenmarx.com/projects/webtools.

Major features in TMA-WC-ProductTargeting include:

* Content & Product Targeting

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. See "TMA WebTools->TMA ProductTargeting"


== Frequently Asked Questions ==

= Can I use the plugin without WebTools? =

No, the plugin is build on the WebTools WordPress integration.
